2-11、比较同一个文件在不同分支的区别

Date：2022-05-22

# 命令格式：git diff 分支1 分支2 -- 文件名

# 比较 index.html 在 dev分支 和 master分支 的区别
git diff dev master -- index.html

如果文件很大，命令行看起来不方便，可以将结果输入到文件中查看

# 将比较结果输入到 diff.txt 中
git diff dev master -- index.html > diff.txt
